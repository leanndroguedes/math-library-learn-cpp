## How to compile
The best way to compile that program is to use [CMake](http://www.cmake.org/).

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Run the app
The output should resemble:

```bash
a + b = 106.4
a - b = -91.6
a * b = 732.6
a / b = 0.0747475
```
